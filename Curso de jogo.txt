Preparação:

1) Embaralhar e distribuir as cartas de personagem até que um jogador receba a carta de Mediador. O jogador que recebeu essa carta se torna o Mediador da primeira rodada.

2) Reembaralhe as cartas de personagem e dê uma para cada jogador. Alternativamente, deixe que os jogadores escolham o proprio personagem, começando pelo Mediador e continuando em sentido horario.

3) Deixe as cartas de personagem restantes de lado e embaralhe as cartas restantes.

Jogo:

1.1) ( Caso seja a primeira rodada ) Retire consecutivamente as cartas até que se tenha um local, um ferimento e uma arma. A primeira carta de cada tipo que ser retirada será a informação inicial do corpo. Depois disso, embaralhe as restantes.

1.2) ( Caso não seja a primeira rodada ) O mediador olha o baralho de pistas e escolhe um local, um ferimento e uma arma para serem usadas como informação inicial. O resto das cartas são embaralhadas.

2) As pistas restantes são distribuidas entre os jogadores, até que todos os jogadores tenham 5 cartas cada, exceto testemunhas que devem ter 2 cartas cada.

3) O mediador começa acusando algum outro jogador, podendo livremente usar as informações e/ou as cartas de sua mão.

4.1) O jogador acusado se defende, usando as cartas de sua mão para acusar algum jogador com exceção de testemunhas. Volte para ( 4 ).

4.2) ( Caso o jogador não consiga se defender ) Algum outro jogador defende o acusado, usando suas proprias cartas para acusar algum outro jogador com exceção de testemunhas. Volte para ( 4 ).

4.3) ( Caso o jogador não consiga se defender e ninguém o defenda ) O jogador acusado é condenado.

5) O jogador condenado se torna uma testemunha, e o jogador que o acusou se torna o Mediador da proxima rodada.

6) As pistas são recolhidas e embaralhadas. Uma nova rodada começa.
